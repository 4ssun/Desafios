O que é um Glossário?

Glossário é um dicionário que busca explicar o significado de termos pouco conhecidos ou arcáicos

Então, como criar um glossário em HTML5?

Você precisa ter em mente três termos para construção de um Glossário: dl,dt e dd.

Definition List (dl)

É como você vai iniciar o glossário, a Tag dl é a abertura de uma lista de definições de termos e descrições.

Definition Term (dt)

É o termo que vou buscar dar definição a ele com a tag dd, posso criar vários dt's para serem definidos em uma única dt.

Definition Description (dd)

É onde chegamos na descrição dos termos. Aqui vai entrar toda descrição que busca explicar os itens de uma dt.

Conclusão

Glossário é uma forma mega simples de tornar algum item do seu site descritivo e explicativo com essa tag própria, tendo também como opção a table list. Opção excelente para sites instituicionais ou de tutoriais.
